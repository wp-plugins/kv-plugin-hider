== Description ==

Generally all the plug-ins are allowed to see your logged in users, If you want to hide a particular plugin From your users, Just use this plugin to hide it from the plugins.php page.  But it will work if you activated the plugin.

**Features**

* Easy install
* Easy to show hide from the plugins directory.

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.

**Installation via Wordpress**

1. Go to the menu 'Plugins' -> 'Install' and search for 'WP List PlugIns'
1. Click 'install'

**Manual Installation**

1. Upload folder `kv-plugin-hider` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
