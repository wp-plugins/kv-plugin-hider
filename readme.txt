== Description ==

Generally all the plugins are apear  to your logged in users, if you want to hide a particular plugin to your users, just use this plugin to hide it from the plugins.php page. 

**Features**

* Easy install
* Easy to show hide from the plugins directory.

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.

**Installation via Wordpress**

1. Go to the menu 'Plugins' -> 'Install' and search for 'WP List PlugIns'
1. Click 'install'

**Manual Installation**

1. Upload folder `kv-plugin-hider` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
